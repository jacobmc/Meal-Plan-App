## Gotcha to not repeat

**Never `--delete-branch` the base PR of a stacked pair.** Doing so permanently closes the stacked PR — GitHub refuses to reopen or retarget a closed PR whose base branch was deleted. Correct sequence: retarget the stacked PR to `main` with `gh pr edit <n> --base main` **before** merging the base, then merge each and delete both branches at the end.
